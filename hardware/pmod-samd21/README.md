# PMOD SAMD21

## Misc IO

| Function | SAMD21G18 pin  | Arduino digital pin | Arduino Analog Pin |
|:-------------:|:-----:|:-----:|:-----:|
| Blue LED | PB23 | 31 | - |
| SWCLK | PA30 | 44 | - |
| SWDIO | PA31 | 45 | - |
| USBD- | PA24 | 28 | - |
| USBD+ | PA25 | 29 | - |
